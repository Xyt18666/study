类 与 js 中的类相似，可以通过 构造类来创建 对象，
类 ，同样有属性 和方法 ，定义类，首字母大写


-----------------------------------------
定义一个类：

class Lei:
	color = "red"
	age = 99

	def fn1(item):
		print("" + item)

	def fn2(item):
		print("22" + item)


pass 关键字： 在类里面，表示不作任何事情
------------------------------------------------

使用类：
fns = Lei()    创建了实例类，并赋值给了fns,通过fns 来访问这个实例类

fns.fn1(11111111)

------------------------------------------------
面向对象的特征：

1. 封装：信息隐蔽技术
2. 继承：子类自动共享父类的数据
3. 多态：不同方法 对 不同对象 做出不同的操作
