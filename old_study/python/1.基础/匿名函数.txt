创建 匿名函数：

xx = lambda x,y : y * x + 1

调用名 = lambda 参数 ：执行的操作

xx(9,7)

------------------------------------------------

实用的内置函数：

筛选出 xx 的内容,返回对象，在使用 list 转为 列表: filret(定义规则的fn,[列表参数])

例子：
list(filter(lambda x : x % 2),[9,51,4,2,6,8])



--------------------------

将序列的每一个元素作为函数的参数进行运算加工，直到序列的最后一个，返回新序列：map(函数,序列)

例子：
list(map(lambda x : x * 2),tange(5))
